# 📊 Revenue Analytics Dashboard

![Dashboard Preview](https://raw.githubusercontent.com/mukherjeesourav687/revenue-analytics-dashboard/main/dashboard_preview.png)

An interactive data visualisation dashboard built using Chart.js to analyse revenue, expenses, and profit trends.

---

## 🚀 Live Demo
👉 https://revenue-analytics-dashboard.netlify.app

---

## 📌 Overview

This project focuses on building a dynamic and interactive dashboard to visualise key business metrics.  
It demonstrates how raw data can be transformed into meaningful insights using modern frontend visualisation tools.

---

## ⚙️ Features

- Multiple chart types: Bar, Line, Pie, Doughnut, Radar
- Dynamic chart switching
- Month-based data filtering
- Real-time metrics display (Revenue, Expenses, Profit)
- Responsive UI design
- Light/Dark mode support
- Smooth animations and transitions
